# RMP

**RMP** — это статичный сайт-каталог в стиле стримингового сервиса, который позволяет не только **искать фильмы и сериалы**, но и **смотреть их прямо через интерфейс сайта**.

Проект работает без собственной серверной части и использует внешние API для получения информации о тайтлах, постерах, датах выхода, трейлерах и связанных данных.

---

## Основные возможности

- Поиск фильмов, сериалов, мультфильмов и другого контента
- Просмотр тайтлов прямо на сайте
- Открытие подробной информации о фильме или сериале
- Отображение трейлеров и связанных видео
- Фильтрация по нескольким жанрам одновременно
- Дополнительные фильтры:
  - **Тип**: фильм или сериал
  - **Год выхода**: диапазон **от** и **до**
- Поля ручного ввода года с синхронизацией со слайдерами
- Отображение под каждой обложкой:
  - типа контента
  - полной даты выхода
- Добавление тайтлов в избранное
- Полностью статичная архитектура без backend-части

---

## Что представляет собой проект

RMP — это не просто список карточек с TMDB.

Это интерфейс кино-сайта, где пользователь может:

1. Найти нужный тайтл  
2. Отфильтровать результаты по жанрам, типу и годам  
3. Открыть подробную карточку  
4. Посмотреть трейлеры, связанные ролики и данные о релизе  
5. Запустить просмотр самого фильма или сериала через сайт  

---

## Функционал каталога

### Поиск
Сайт поддерживает поиск по названиям тайтлов с корректной работой как для фильмов, так и для сериалов.

### Жанры
Можно выбрать сразу несколько жанров, чтобы сузить результаты.

### Дополнительные фильтры
После нажатия на кнопку **«Применить доп. фильтры»** применяются:

- **Тип контента**
  - фильм
  - сериал

- **Диапазон годов**
  - от
  - до

Годы можно задавать:
- ползунками
- вручную через текстовые поля

Оба способа синхронизированы между собой:
- двигаешь ползунок — меняется число
- меняешь число — двигается ползунок

Если год **от** равен году **до**, поиск идёт по одному конкретному году.

## Карточки тайтлов

Каждая карточка в каталоге содержит:

- постер
- название
- маленькую подпись с автоопределением:
  - **Фильм**
  - **Сериал**
- полную дату выхода

Это позволяет сразу понимать, что именно перед тобой, ещё до открытия страницы с подробностями.

---

## Просмотр контента

Одна из ключевых особенностей проекта — возможность **смотреть фильмы и сериалы прямо на сайте**, а не просто просматривать информацию о них.

Сайт задуман как полноценный frontend для кино-каталога, где после выбора тайтла пользователь может перейти от поиска к непосредственному просмотру в том же интерфейсе.

Дополнительно в карточке **«Подробнее»** могут отображаться:
- трейлеры
- тизеры
- связанные видео
- описание
- дата релиза
- прочая информация по тайтлу

## Архитектура

Проект остаётся полностью **статичным**, то есть:
- без собственной серверной части
- без базы данных
- без отдельного backend

Вся логика работает на клиентской стороне в браузере.

Это позволяет:
- легко хостить сайт где угодно
- быстро разворачивать проект
- использовать его как готовую основу для кино-платформы
